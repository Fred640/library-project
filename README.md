# 📚 Open Library

**Open Library** — это мой учебный проект, в рамках которого я применил на практике знания **Python/Django**, **JavaScript/React** и **Django REST API**.

## 🚀 О проекте

Сайт представляет собой библиотеку, содержащую все книги из **российской школьной программы**. Пользователи могут читать книги онлайн на сайте или скачивать их на устройство. Также реализована возможность загрузки собственных книг и конспектов в открытый доступ для других пользователей.

## ✨ Функционал

### 📖 Для всех пользователей:
- Чтение и скачивание книг из школьной программы
- Доступ к материалам, загруженным другими пользователями
- Добавление авторов и книг в избранное


### 👨‍💻 Для зарегистрированных пользователей (2 этап):
- Публикация собственных книг и конспектов в общий доступ

## 🔐 Система регистрации

На сайте реализована **двухэтапная регистрация**:

| Этап | Возможности |
|------|-------------|
| **1 этап** | Чтение и скачивание книг, добавление в избранное |
| **2 этап** | Публикация своих материалов |
